Lesson 2: Footprinting & Reconnaissance

Welcome to the second lesson of your Ethical Hacking journey. In this lesson, you'll learn how to perform footprinting and reconnaissance on a target.

üõ†Ô∏è What is Footprinting?

Footprinting is the process of gathering information about a target system or network. This includes identifying IP addresses, domain names, and organizational details.

üîç Types of Footprinting

  • Passive Footprinting: Collecting information without interacting with the target (e.g., OSINT).
  • Active Footprinting: Interacting with the target to gather details (e.g., port scanning).

üåê Tools for Footprinting

  • WHOIS: Fetch domain registration details.
  • Nslookup: Resolve domain names to IPs.
  • Nmap: Network scanning and service enumeration.
  • Shodan: Search engine for devices and services.

1Ô∏è‚É£ WHOIS Lookup (Python)

Use this Python script to fetch WHOIS information:

import whois

# Enter the target domain
domain = input("Enter domain: ")

# Fetch WHOIS details
w = whois.whois(domain)

print("\n--- WHOIS Information ---")
print(f"Domain: {w.domain_name}")
print(f"Registrar: {w.registrar}")
print(f"Creation Date: {w.creation_date}")
print(f"Expiration Date: {w.expiration_date}")
print(f"Name Servers: {w.name_servers}")
