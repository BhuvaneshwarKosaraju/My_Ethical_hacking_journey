Lesson 6: Vulnerability Scanning

Vulnerability scanning is a key component of ethical hacking, used to identify security flaws in the target system. This lesson focuses on using tools like Nessus and OpenVAS to detect vulnerabilities.

📘 Tools for Vulnerability Scanning

  • Nessus: A widely used vulnerability scanner for detecting weaknesses in systems.
  • OpenVAS: An open-source alternative to Nessus, used for vulnerability management and scanning.

💻 Hands-on Labs

1️⃣ Using Nessus for Vulnerability Scanning

Install Nessus and perform a vulnerability scan using the following command:

nessus -q -T HTML -o report.html

This will generate an HTML report of vulnerabilities found in the system.

2️⃣ Using OpenVAS for Vulnerability Scanning

Here’s how to perform a vulnerability scan using OpenVAS:

openvas-check-setup --v9

🔥 Challenge

Run a vulnerability scan using Nessus on your local machine and generate a report.
