üîç Introduction to Reconnaissance

Reconnaissance (or recon) is the first phase of ethical hacking. It involves gathering information about the target before launching any attacks.

üéØ **Objectives of Reconnaissance:**

  • Identify target domains and subdomains.
  • Discover IP addresses, ports, and services.
  • Enumerate users and emails.
  • Collect data for further exploitation.

üõ†Ô∏è **Python Script: WHOIS Lookup**

Below is a Python script to automate WHOIS lookups.

import whois

def whois_lookup(domain):
    try:
        w = whois.whois(domain)
        print("Domain:", w.domain_name)
        print("Registrar:", w.registrar)
        print("Creation Date:", w.creation_date)
        print("Expiration Date:", w.expiration_date)
    except Exception as e:
        print("Error:", e)

# Example usage
whois_lookup("example.com")
